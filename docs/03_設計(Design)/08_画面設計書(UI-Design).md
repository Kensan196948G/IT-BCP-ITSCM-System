## 1. 概要

### 1.1 デザイン方針

| 方針 | 説明 |
|------|------|
| 情報優先 | 災害時に最も重要な情報を最短で到達できるUI |
| 視認性 | RTO状態を色分け + アイコン + テキストで3重に表現 |
| 操作性 | 3クリック以内で主要操作に到達 |
| レスポンシブ | PC/タブレット/スマートフォンすべてに最適化 |
| ダークモード | 長時間監視時の目の負担軽減（ダークモード対応） |
| アクセシビリティ | WCAG 2.1 AA準拠、キーボード操作完全対応 |

## 2. デザインシステム

### 2.1 カラーパレット

| 名前 | カラーコード | 用途 |
|------|------------|------|
| Primary | #1E40AF (青) | メインアクション、リンク、ナビゲーション |

| Info / Recovered | #3B82F6 (青) | RTO復旧済、情報 |

| Background (Dark) | #0F172A | ダークモード背景 |

| 要素 | フォント | サイズ | ウェイト |
|------|---------|--------|---------|

| H2（セクションタイトル） | Noto Sans JP | 20px / 1.25rem | SemiBold (600) |
| H3（カードタイトル） | Noto Sans JP | 16px / 1rem | SemiBold (600) |

| 数値（中） | JetBrains Mono | 20px / 1.25rem | SemiBold (600) |

### 2.3 アイコンシステム（RTO状態）

| 状態 | 色 | アイコン | テキスト |
|------|-----|---------|---------|
| On Track | 緑 (#22C55E) | ● (塗り丸) + チェック | 順調 |
| At Risk | 黄 (#EAB308) | ▲ (三角) + 警告 | 警告 |
| Overdue | 赤 (#EF4444) | ✕ (バツ丸) | 超過 |
| Recovered | 青 (#3B82F6) | ● (塗り丸) + ダブルチェック | 復旧済 |
| Not Started | グレー (#6B7280) | ○ (空丸) + 時計 | 未開始 |
