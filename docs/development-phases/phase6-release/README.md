## 4. ロールバック計画

```
ロールバック判断基準:
  - デプロイ後 30 分以内に Critical エラーが発生
  - ヘルスチェックが 3 回連続で失敗
  - 主要 API の応答時間が 5 秒を超える

ロールバック手順 (目標: 15 分以内):
  1. Azure Front Door でトラフィックを前バージョンに切替
  2. Azure Container Apps を前イメージに戻す
  3. DB マイグレーションの downgrade（必要な場合）
  4. ロールバック完了通知
  5. 原因調査・修正計画立案
```
