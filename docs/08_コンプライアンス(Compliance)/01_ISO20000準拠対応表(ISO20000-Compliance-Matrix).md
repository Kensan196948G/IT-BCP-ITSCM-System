## 1. 本文書の目的

本文書は、IT-BCP-ITSCMシステムがISO/IEC 20000-1:2018（ITサービスマネジメントシステム）の要求事項に対してどのように準拠しているかを体系的にマッピングしたものである。特にITサービス継続管理（ITSCM）に関連する条項を中心に、各条項に対する本システムの対応機能、準拠ステータス、およびエビデンス（証跡）の生成方法を詳細に記載する。

## 2. 適用範囲

- 対象組織: みらい建設工業株式会社
- 対象システム: IT事業継続管理システム（IT-BCP-ITSCM-System）
- 対象規格: ISO/IEC 20000-1:2018 全条項（特にITサービス継続管理関連）

## 3. 準拠ステータスの定義

| ステータス | 説明 |
|-----------|------|
| 対応済 | 本システムにて機能実装済みであり、エビデンスが自動生成される |
| 一部対応済 | 機能の一部が実装済みであり、追加の運用手順で補完する |
| 対応予定 | 今後の開発計画に含まれている |
| 対象外 | 本システムの適用範囲外（組織的対応が必要） |

## 4. ISO20000-1:2018 条項別対応表

### 4.1 条項4: 組織の状況

#### 4.1 組織及びその状況の理解

| 項目 | 内容 |
|------|------|
| 条項番号 | 4.1 |
| 要求事項 | 組織は、その目的に関連し、かつSMSの意図した成果を達成する組織の能力に影響を与える外部及び内部の課題を決定しなければならない |
| 本システムの対応 | BIA（業務影響度分析）機能により、業務プロセスへの影響評価、財務影響試算、規制違反リスク評価を実施。建設業特有の外部環境（地震・台風等の自然災害、サプライチェーン途絶）を考慮した分析テンプレートを提供 |
