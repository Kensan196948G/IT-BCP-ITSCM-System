## 使用方法

本チェックリストは、各リリースごとにコピーして使用する。各項目のチェックボックスにチェックを入れ、確認者名と確認日時を記録すること。全ての必須項目にチェックが入るまでリリースを実行してはならない。

**リリースバージョン**: v_____
