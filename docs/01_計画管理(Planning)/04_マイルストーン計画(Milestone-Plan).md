## 1. マイルストーン概要

本計画は、IT-BCP-ITSCM-Systemプロジェクトにおける主要マイルストーンを定義し、各マイルストーンの判定基準、レビューポイント、成果物を明確にするものである。

### 1.1 マイルストーンタイムライン

```
2026年
 4月     5月     6月     7月     8月     9月     10月    11月
  |       |       |       |       |       |       |       |
  M0     M1-1    M1-4    M2-1    M2-2    M2-4    M3-2    M4-3
  PJ     環境    P1      BCP     RTO    MVP/P2   PWA    本番
  開始   Ready   完了    完了    完了    完了     完了   リリース
         M1-2            M2-3                    M3-1    M4-1
         Azure           訓練                   ダッシュ  DR
         Ready           管理                   ボード   完了
         M1-3                                   M3-3    M4-2
         認証                                    P3     テスト
         完了                                   完了    完了
                                                        M4-4
                                                        PJ完了
```

## 3. マイルストーン判定プロセス

### 3.1 判定フロー

```
1. PM: 判定基準チェックリストの確認・記入
      ↓
2. PM: 関連成果物の収集・整理
      ↓
3. チームリード: 各領域の判定基準レビュー
      ↓
4. PM: マイルストーン判定会議の招集
      ↓
5. 判定会議: 判定基準の合否判定
      ↓
6a. 全基準合格 → マイルストーン達成・次フェーズ開始
6b. 不合格あり → 是正措置計画策定 → 再判定日設定
```

### 3.2 エスカレーション基準

| 状況 | エスカレーション先 | アクション |
|------|-----------------|----------|
| マイルストーン遅延1週間以内 | PM | リカバリー計画策定 |
| マイルストーン遅延2週間以上 | プロジェクトオーナー | スコープ/スケジュール調整判断 |
| 判定基準の重大な未達 | プロジェクトオーナー | Go/No-Go判定 |
| フェーズゲートのNo-Go | IT統括本部長 | プロジェクト継続判断 |
